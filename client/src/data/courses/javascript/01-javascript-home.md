# Welcome to the JavaScript Tutorial

JavaScript is the language of the web. It runs in every browser on Earth, powers the back end of Netflix and PayPal through Node.js, drives mobile apps with React Native, controls satellites at NASA, and ships in everything from smart TVs to spacecraft. If you write code that humans interact with today, the odds are overwhelming that some of it is JavaScript.

## What you will learn

This course takes you from absolute beginner to confident, modern JavaScript developer. Every lesson is short, focused on one idea, and ends with code you can paste into your browser console or a `.js` file and run. By the end you will be able to:

- Run JavaScript in the browser, in Node.js, and in your editor.
- Use every fundamental data type — numbers, strings, booleans, arrays, objects, `Map`, `Set`, `Symbol`.
- Write clean control flow with `if`, `switch`, `for`, `for…of`, `while`, and array methods.
- Define your own functions, arrow functions, classes, and ES modules.
- Master the parts of the language that trip up beginners: `this`, hoisting, closures, the prototype chain, `==` vs `===`.
- Handle errors, parse JSON, match regular expressions, and work with dates.
- Write asynchronous code with callbacks, promises, and `async`/`await`.
- Manipulate the DOM, listen to events, and call HTTP APIs with `fetch`.
- Test your code with Vitest and follow modern best practices.

## How the course is organized

Lessons are ordered for first-time learners — start at the top and work your way down. Each lesson introduces one concept, shows it in action, and warns you about the most common pitfalls. Use the sidebar to jump around once you have the basics.

## Prerequisites

None. You don't need any prior programming experience. If you can use a text editor and a web browser, you're ready.

## Try It Yourself

Throughout the course you'll see code blocks like the one below. Open your browser, press **F12** to open developer tools, click the **Console** tab, paste the code, and press **Enter**.

```javascript
console.log("Hello, World!");
```

That single line is a complete JavaScript program. No imports, no `main` function, no compilation step — JavaScript is famous for letting you start instantly.
