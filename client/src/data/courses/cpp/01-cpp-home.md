# Welcome to the C++ Tutorial

C++ is a powerful, general-purpose programming language built on top of C. It adds **classes, objects, templates, exceptions, the Standard Template Library (STL)**, and many other features that make it one of the most widely used languages in the world.

You will find C++ behind:

- High-performance applications: **game engines** (Unreal, Unity's core), browsers (Chrome), databases (MySQL, MongoDB).
- Operating systems and embedded firmware.
- Competitive programming, where speed and rich data structures matter.
- Trading systems, image/video processing, and scientific computing.

## What you will learn

By the end of this course you will be comfortable with:

1. Writing, compiling, and running C++ programs.
2. Variables, primitive types, and operators.
3. Strings, arrays, and the STL `vector` / `string`.
4. Control flow: `if`, `switch`, `while`, `for`, `break`, `continue`.
5. Functions, references, pointers, and lambdas.
6. Object-oriented programming: classes, constructors, inheritance, polymorphism.
7. Templates, the STL, and modern C++ idioms (C++11 → C++20).
8. Exceptions, file I/O, and basic memory management.

## Why learn C++?

- **Performance**: Compiled to native code, with zero-cost abstractions.
- **Control**: Direct access to memory and hardware when you need it.
- **Versatility**: Procedural, object-oriented, generic, and functional styles all in one language.
- **Foundation**: Almost every modern language borrows from C++ — learning it makes everything else easier.
