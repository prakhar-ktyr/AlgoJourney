# Types of Cyber Threats

Understand the different categories of cyber threats and who's behind them.

---

## Threat Categories

| Category | Description | Example |
|----------|-------------|---------|
| **Malware** | Malicious software | Ransomware, viruses, trojans |
| **Social Engineering** | Manipulating humans | Phishing, pretexting |
| **Network Attacks** | Exploiting network weaknesses | DDoS, MitM, sniffing |
| **Web Attacks** | Targeting web applications | SQL injection, XSS |
| **Insider Threats** | From within the organization | Disgruntled employees |
| **Advanced Persistent Threats** | Long-term targeted attacks | Nation-state espionage |

---

## Threat Actors

Who carries out attacks?

### Script Kiddies
- Low skill, use pre-built tools
- Motivation: Fun, bragging rights
- Risk level: Low to medium

### Hacktivists
- Politically or socially motivated
- Examples: Anonymous, LulzSec
- Targets: Governments, corporations
- Tactics: DDoS, defacement, data leaks

### Cybercriminals
- Financially motivated
- Organized groups operating like businesses
- Tactics: Ransomware, fraud, data theft
- Revenue: Billions annually through the dark web

### Nation-State Actors
- Government-sponsored
- Highly sophisticated, well-funded
- Targets: Critical infrastructure, military, intellectual property
- Examples: APT28 (Russia), APT41 (China), Lazarus (North Korea)

### Insider Threats
- Current or former employees, contractors
- Have legitimate access
- Can be malicious or accidental
- Often the hardest to detect

## Attack Motivations

| Motivation | Actor Type | Example |
|-----------|-----------|---------|
| Financial gain | Cybercriminals | Ransomware demanding Bitcoin |
| Espionage | Nation-states | Stealing defense secrets |
| Disruption | Hacktivists | DDoS on government sites |
| Revenge | Insiders | Deleting databases after being fired |
| Fame | Script kiddies | Defacing a popular website |
| Warfare | Nation-states | Attacking power grids |

## Emerging Threats

### AI-Powered Attacks
- Deepfake audio/video for social engineering
- AI-generated phishing emails (harder to detect)
- Automated vulnerability discovery

### Supply Chain Attacks
- Compromise a trusted vendor to reach their customers
- Example: SolarWinds hack (2020) — affected 18,000 organizations
- Example: Log4Shell (2021) — vulnerability in a logging library

### IoT Attacks
- Smart devices with weak security
- Botnets using compromised cameras, routers
- Example: Mirai botnet DDoS attack

### Ransomware-as-a-Service (RaaS)
- Criminals sell ransomware toolkits
- Anyone can launch ransomware attacks without technical skill
- Revenue-sharing model between developers and affiliates

## The Cyber Kill Chain

Lockheed Martin's model of how attacks unfold:

```
1. Reconnaissance    → Research the target
2. Weaponization     → Create malicious payload
3. Delivery          → Send to target (email, web, USB)
4. Exploitation      → Trigger the vulnerability
5. Installation      → Install backdoor/malware
6. Command & Control → Establish remote access
7. Actions on Objectives → Steal data, encrypt files, etc.
```

Defenders can break the chain at any stage.

## Threat Landscape Statistics

- **560,000** new malware samples detected daily
- **36%** of data breaches involve phishing
- **82%** of breaches involve the human element
- Ransomware attacks occur every **11 seconds**
- Average time to identify a breach: **204 days**

---

## Key Takeaways

- Threats range from simple (script kiddies) to sophisticated (nation-states)
- Motivations include money, politics, espionage, and revenge
- Emerging threats include AI attacks, supply chain compromise, and IoT botnets
- The **Cyber Kill Chain** helps understand and disrupt attack stages
- Understanding threats is the first step to defending against them
