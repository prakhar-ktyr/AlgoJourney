# What is Cybersecurity

Cybersecurity is the practice of protecting systems, networks, and data from digital attacks, unauthorized access, and damage.

---

## Definition

**Cybersecurity** (also called information security or IT security) encompasses the technologies, processes, and practices designed to protect:

- **Computers** and servers
- **Networks** (local and internet)
- **Data** (personal, financial, business)
- **Software** and applications
- **Mobile devices**
- **Cloud infrastructure**

---

## The CIA Triad

The three core principles of cybersecurity:

```
        Confidentiality
             /\
            /  \
           /    \
          /      \
         /________\
  Integrity      Availability
```

1. **Confidentiality** — Only authorized people can access data
2. **Integrity** — Data is accurate and hasn't been tampered with
3. **Availability** — Systems and data are accessible when needed

We'll explore each in depth in the next lesson.

## Types of Cybersecurity

| Domain | What It Protects |
|--------|-----------------|
| **Network Security** | Computer networks from intruders |
| **Application Security** | Software from threats during development and after deployment |
| **Information Security** | Data integrity and privacy (stored and in transit) |
| **Cloud Security** | Cloud-based systems, data, and infrastructure |
| **Endpoint Security** | Devices like laptops, phones, and tablets |
| **Operational Security** | Processes for handling and protecting data |
| **Identity Security** | User identities and access permissions |
| **Disaster Recovery** | Responding to and recovering from incidents |

---

## Common Cyber Threats

A quick overview (each covered in detail later):

- **Malware** — Malicious software (viruses, ransomware, trojans)
- **Phishing** — Fake emails/sites to steal credentials
- **Man-in-the-Middle** — Intercepting communications
- **Denial of Service** — Overwhelming systems with traffic
- **SQL Injection** — Inserting malicious code into databases
- **Zero-Day Exploits** — Attacking unknown vulnerabilities

---

## Who Needs Cybersecurity?

Everyone. But especially:

- **Businesses** — Protect customer data, intellectual property, operations
- **Governments** — National security, citizen data, critical infrastructure
- **Healthcare** — Patient records (HIPAA compliance)
- **Financial institutions** — Transaction security, fraud prevention
- **Individuals** — Personal data, identity theft protection

---

## Cybersecurity vs. Information Security

| | Cybersecurity | Information Security |
|---|---|---|
| **Scope** | Digital/cyber threats | All information (digital + physical) |
| **Focus** | Protecting systems from attacks | Protecting data in any form |
| **Example** | Preventing a DDoS attack | Securing paper documents in a locked cabinet |

Cybersecurity is a **subset** of information security focused specifically on digital threats.

---

## Key Roles in Cybersecurity

| Role | Responsibility |
|------|---------------|
| Security Analyst | Monitor and respond to threats |
| Penetration Tester | Find vulnerabilities by simulating attacks |
| Security Engineer | Build and maintain security systems |
| Security Architect | Design secure system architectures |
| CISO | Chief Information Security Officer — leads security strategy |
| Incident Responder | Handle active security breaches |

---

## Key Takeaways

- Cybersecurity protects systems, networks, and data from digital threats
- It's built on **Confidentiality, Integrity, and Availability** (CIA)
- Covers multiple domains: network, application, cloud, endpoint, and more
- Every organization and individual needs cybersecurity
- The field offers diverse, in-demand career paths
