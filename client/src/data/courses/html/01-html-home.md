# HTML Tutorial

HTML is the standard markup language for creating web pages. It stands for **HyperText Markup Language**.

Every website you visit — from Google to GitHub — is built with HTML at its core. HTML describes the **structure** of a web page using a series of elements that tell the browser how to display content.

## What is HTML?

- HTML stands for **H**yper**T**ext **M**arkup **L**anguage
- HTML is the **standard markup language** for creating web pages
- HTML describes the **structure** of a web page
- HTML consists of a series of **elements**
- HTML elements tell the browser **how to display** the content

> [!NOTE]
> HTML is not a programming language — it is a *markup language*. It defines the structure and meaning of content, while CSS handles appearance and JavaScript handles behavior.

---

## A Simple HTML Document

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>
```

### What does this code do?

- `<!DOCTYPE html>` declares that this is an HTML5 document
- `<html>` is the root element of the page
- `<head>` contains metadata (information about the page)
- `<title>` sets the title shown in the browser tab
- `<body>` contains all the visible content
- `<h1>` defines a large heading
- `<p>` defines a paragraph
