# Welcome to the Node.js Tutorial

Node.js took JavaScript out of the browser and put it everywhere. It runs the back end of Netflix, LinkedIn, Uber, NASA mission control, and millions of APIs that power the modern web. If a company has a web product, there is an excellent chance Node.js is somewhere in the stack.

## What you will learn

This course takes you from absolute beginner to confident, production-ready Node.js developer. Every lesson is hands-on with copyable code you can run on your own machine. By the end you will be able to:

- Install Node.js and run scripts from the terminal.
- Understand the event-driven, non-blocking architecture that makes Node.js fast.
- Use core modules — `fs`, `path`, `os`, `events`, `http`, `url`, `crypto`, `streams`, `buffers`.
- Master asynchronous programming: callbacks, promises, `async`/`await`, and the event loop.
- Build web servers and REST APIs with Express.
- Connect to databases (MongoDB with Mongoose, MySQL).
- Implement authentication with JWT, sessions, and password hashing.
- Handle errors, debug, and write automated tests.
- Scale with worker threads, clustering, and WebSockets.
- Deploy to production with PM2, Docker, and environment management.

## How the course is organized

Lessons are ordered for first-time learners — start at the top and work your way down. Each lesson introduces one concept, shows it in action, and warns you about the most common pitfalls. Use the sidebar to jump around once you have the basics.

## Prerequisites

Basic JavaScript knowledge is helpful but not required. If you can write a `for` loop and call a function in JavaScript, you are ready. If not, check out our JavaScript tutorial first.

## Try It Yourself

Throughout the course you will see code blocks like the one below. Once Node.js is installed (next few lessons), open your terminal, create a file, and run it:

```javascript
console.log("Hello from Node.js!");
```

```bash
node hello.js
```

That is a complete Node.js program. No browser, no HTML file, no build step — just JavaScript running directly on your operating system.
