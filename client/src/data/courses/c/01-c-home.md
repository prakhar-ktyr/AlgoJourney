# Welcome to the C Tutorial

C is one of the most influential programming languages ever created. Operating systems (Linux, Windows kernels, macOS internals), embedded firmware, databases, compilers, and even the runtimes of higher-level languages like Python and JavaScript are written largely in C. Learning C teaches you how computers _actually work_ — memory, pointers, the stack, the heap — knowledge that pays off in every other language you'll ever use.

## What you will learn

This course takes you from absolute beginner to confidently writing real C programs. Every lesson is hands-on with copyable examples you can run on your own machine. By the end you will be able to:

- Set up a C compiler and build & run programs from the command line.
- Use every fundamental data type, operator, and control-flow construct.
- Write your own functions, structures, and modular multi-file programs.
- Manipulate memory directly with pointers, `malloc`, and `free`.
- Read from and write to files.
- Debug crashes and reason about undefined behavior.

## How the course is organized

The lessons are ordered for first-time learners — start at the top and work your way down. Each lesson is short, focused on one concept, and ends with a runnable example. Use the sidebar on the left to jump around once you have the basics.

## Prerequisites

None! You don't need any prior programming experience. If you can use a text editor and a terminal, you're ready.

## Try It Yourself

Throughout the course you'll see code blocks like the one below. Don't worry if it looks unfamiliar — the very next lessons walk you through installing a C compiler, creating your first file, and running it from the terminal step by step.

```c
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```

For now, just read the example to get a feel for the syntax. By the end of the **Get Started** lesson you'll be running this exact program on your own machine.
