# Welcome to the Java Tutorial

Java is one of the most widely used programming languages in the world. It powers everything from Android apps and enterprise back-end systems to financial trading platforms, scientific tools, and big-data frameworks like Hadoop and Spark. Three things make Java special:

1. **Write once, run anywhere.** Java code is compiled to bytecode that runs on the Java Virtual Machine (JVM), so the same `.class` file works on Linux, macOS, and Windows.
2. **Strong, static typing.** The compiler catches a large class of bugs before the program even starts.
3. **A massive standard library and ecosystem.** Collections, networking, concurrency, file I/O, JSON, web servers — there is a battle-tested library for almost everything.

## What you will learn

This course takes you from absolute beginner to writing real, idiomatic Java. By the end you will be able to:

- Install a JDK, write Java source files, compile them, and run them from the terminal.
- Use every fundamental data type, operator, and control-flow construct.
- Build classes, interfaces, and full object-oriented designs using inheritance, polymorphism, and encapsulation.
- Handle errors with exceptions, work with files, and parse text with regular expressions.
- Use the Java Collections Framework (`ArrayList`, `HashMap`, `HashSet`, …) effectively.
- Write modern Java with generics, lambdas, streams, `Optional`, and the new date/time API.
- Reason about threads, concurrency, and how the JVM manages memory.
