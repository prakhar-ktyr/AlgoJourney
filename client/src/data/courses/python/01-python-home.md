# Welcome to the Python Tutorial

Python is the most widely-taught programming language in the world. It powers data science at Netflix, machine learning at OpenAI, the build system at Google, the deployment scripts at every modern startup, and the homework of millions of first-year computer science students. Its syntax reads like English, its standard library is huge, and a single laptop is enough to start.

## What you will learn

This course takes you from absolute beginner to confident, idiomatic Python. Every lesson is hands-on with copyable examples you can run on your own machine. By the end you will be able to:

- Install Python and run scripts from the terminal or an editor.
- Use every fundamental data type: numbers, strings, lists, tuples, sets, and dictionaries.
- Write clean control flow with `if`, `for`, `while`, and comprehensions.
- Define your own functions, classes, and modules — and structure a multi-file project.
- Read and write files, parse JSON, match regular expressions, and work with dates.
- Use type hints, virtual environments, `pip`, and write unit tests like a professional.
- Understand iterators, generators, decorators, and `async`/`await`.

## How the course is organized

Lessons are ordered for first-time learners — start at the top and work your way down. Each lesson is short, focused on one concept, and ends with a runnable example. Use the sidebar on the left to jump around once you have the basics.

## Prerequisites

None. You don't need any prior programming experience. If you can use a text editor and a terminal, you're ready.

## Try It Yourself

Throughout the course you'll see code blocks like the one below. Don't worry if it looks unfamiliar — the very next lessons walk you through installing Python, creating your first file, and running it from the terminal step by step.

```python
print("Hello, World!")
```

That single line is a complete Python program. No imports, no `main` function, no semicolons — Python is famous for getting out of your way.
